# CleanTemplate

A clean, production-ready iOS starter template built with SwiftUI and VIPER architecture.

Based on [SwiftfulStarterProject](https://github.com/SwiftfulThinking/SwiftfulStarterProject) by Swiftful Thinking, LLC, licensed under MIT.

## Features

- VIPER + RIBs architecture pattern
- Three build configurations (Mock, Dev, Production)
- Firebase integration (Auth, Firestore, Analytics, Crashlytics)
- RevenueCat for in-app purchases
- Mixpanel analytics
- Gamification system (Streaks, XP, Progress)
- SwiftfulThinking package ecosystem

## Getting Started

1. Clone this repository
2. Install Xcode templates from [SwiftfulThinking/XcodeTemplates](https://github.com/SwiftfulThinking/XcodeTemplates)
3. Run `rename_project.sh` to customize the project name
4. Configure your Firebase and RevenueCat credentials
